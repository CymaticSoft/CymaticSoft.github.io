RvtVa3c - Revit Three.js Exporter

RvtVa3c is an add-in for Autodesk Revit that exports the Revit BIM model to a three.js version 4 JSON format.

RvtVa3c is part of the vA3C project, "an open source, browser-based 3D model viewer for AEC models that uses three.js to render 3D geometry in the browser, enabling authors in the AEC industry to easily publish their 3D design work on the web for free.

RvtVa3c and its documentation live in their own RvtVa3c GitHub repository. Please refer to that for all further details.

Good luck and have fun with three.js!
